Zed City API Documentation

Welcome to the Zed City API! This API provides access to all game data for the zombie survival game Zed City. Use these endpoints to retrieve information about quests, player activities, character progression, world state, and more.

Base URL: https://api.zed.city

Quick Start

All endpoints are simple GET requests that return JSON data. No authentication is required for public game data.

Example Request
curl -X GET "https://api.zed.city/getQuests" \
  -H "Content-Type: application/json"

Response Format

All responses are returned in JSON format with appropriate HTTP status codes.

Authentication

Currently, the Zed City API endpoints listed here are public and do not require authentication. All game data can be accessed without API keys or tokens.

Public Access: No authentication required for current endpoints.

Future endpoints may implement authentication for player-specific data or administrative functions.

Error Handling

The API uses conventional HTTP response codes to indicate success or failure of requests.

HTTP Status Codes

Code Description
200 OK - Request successful
400 Bad Request - Invalid parameters
404 Not Found - Endpoint or resource doesn't exist
500 Internal Server Error

Quests

Retrieve information about available quests, missions, and objectives in Zed City.

GET /getQuests
Get a list of all available quests in the game, including quest details, requirements, rewards, and completion status.
Example Response
{
  "quests": [
    {
      "id": 1,
      "title": "First Steps",
      "description": "Learn the basics of survival in Zed City",
      "type": "tutorial",
      "rewards": {
        "experience": 100,
        "items": ["basic_weapon", "food_ration"]
      },
      "requirements": {
        "level": 1,
        "completed_quests": []
      },
      "status": "available"
    }
  ]
}

Activities

Access information about various survival activities available to players.

GET /getHunting
Get hunting locations, available prey, success rates, and hunting equipment requirements.
GET /getScavenge
Retrieve scavenging opportunities, loot tables, location safety levels, and resource availability.
GET /getExplore
Get exploration data including new areas to discover, exploration risks, and potential discoveries.

Character

Manage character progression, skills, and abilities.

GET /getSkills
Retrieve the complete skill tree, skill descriptions, prerequisites, and progression paths.
GET /getPerks
Get available perks, their effects, unlock requirements, and perk categories.
GET /getStats
Access character statistics, attributes, health, stamina, and other vital information.

World

Explore the game world, locations, and environmental data.

GET /getLocation
Get current location data, nearby points of interest, safety levels, and available actions.
GET /getStronghold
Retrieve stronghold information including defenses, upgrades, capacity, and resource storage.
GET /getSettings
Access game settings, configuration options, and player preferences.
GET /loadItems
Load item database including all weapons, armor, consumables, and crafting materials.

Factions

Manage faction relationships, membership, and social interactions.

GET /getFaction
Get information about all available factions, their ideologies, benefits, and requirements.
GET /getFactionMembers
Retrieve faction membership lists, member ranks, and contribution statistics.
GET /getUsers
Access user directory, player profiles, online status, and public statistics.

Combat

Combat system data, fight mechanics, and battle logs.

GET /getFight
Get current fight data, combat options, enemy information, and battle state.
GET /getFightLog
Retrieve combat history, battle outcomes, damage dealt/received, and fight statistics.

Communication

Radio systems and communication networks in the wasteland.

GET /getRadioTower
Get list of all radio towers, their status, broadcast range, and available channels.
GET /getRadioTower?id=${radioId}
Get detailed information about a specific radio tower by ID, including broadcasts and signal strength.

Query Parameters

Parameter Type Required Description
id string/integer required Radio tower ID to retrieve

Trading

Economy, markets, and trading systems.

GET /getTrades
Get available trades, trade offers, and player-to-player trading opportunities.
GET /getMarket
Access market data, item prices, supply and demand, and trading volume statistics.
GET /getStore?store_id=junk
Get store inventory, item availability, prices, and store-specific information.

Query Parameters

Parameter Type Required Description
store_id string required Store identifier (e.g., "junk", "weapons", "medical")

Crafting

Item crafting, recipes, and manufacturing systems.

GET /getRecipes?type=${type}
Retrieve crafting recipes filtered by type, including required materials, crafting time, and success rates.

Query Parameters

Parameter Type Required Description
type string required Recipe category (e.g., "weapons", "armor", "consumables", "tools")
Example Response
{
  "recipes": [
    {
      "id": 1,
      "name": "Makeshift Knife",
      "type": "weapons",
      "materials": [
        {
          "item": "scrap_metal",
          "quantity": 2
        },
        {
          "item": "cloth",
          "quantity": 1
        }
      ],
      "crafting_time": 300,
      "success_rate": 85,
      "skill_required": {
        "crafting": 1
      }
    }
  ]
}
